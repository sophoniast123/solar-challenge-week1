# Solar Challenge — Week 1

Project repository for Week 1 of the Solar Challenge coursework. This repository contains data, exploratory notebooks, and scripts used to analyse financial news and related datasets.

**Contents**
- `data/` : Raw and processed datasets used by the notebooks and scripts.
- `notebooks/` : Jupyter notebooks containing EDA and analysis. The primary notebook is `01_EDA_Financial_News.ipynb`.
- `scripts/` : Helper scripts and small utilities.
- `src/` : Python package source code for reusable modules (if any).
- `tests/` : Unit tests for functions in `src/`.

**Getting Started**

Prerequisites:

- Python 3.8+ installed
- Recommended: create and activate a virtual environment

Install dependencies:

```
pip install -r requirements.txt
```

Running the notebooks:

- Open `notebooks/01_EDA_Financial_News.ipynb` with Jupyter or VS Code's notebook support.

Quick project structure overview:

- `data/newsData/raw_analyst_ratings.csv` — example raw dataset included for week 1.
- `notebooks/01_EDA_Financial_News.ipynb` — performs exploratory data analysis on available news and ratings data.

How to run tests:

```
python -m pytest -q
```
